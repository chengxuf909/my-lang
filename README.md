# AI学习小项目

## 环境安装

- python3.13.3
- uv (python的包管理工具，感觉比pip还有venv更好用)
    ```
    uv install
    uv init
    uv add xxx
    ```
- langchain (langchan相关的包一大堆)
    ```
    uv add langchain
    ```
